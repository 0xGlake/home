# Movie Images

## Naming Convention
All movie poster images should follow this naming pattern:
- Lowercase with hyphens
- No special characters except hyphens
- File extension: `.jpg` or `.webp`

## File List
Place your movie poster images with these exact filenames:

1. `ghost-in-the-shell.jpg`
2. `gattaca.jpg`
3. `spirited-away.jpg`
4. `howls-moving-castle.jpg`
5. `iron-man-1.jpg`
6. `james-bond-casino-royale.jpg`
7. `whiplash.jpg`
8. `the-princess-bride.jpg`
9. `catch-me-if-you-can.jpg`
10. `ex-machina.jpg`
11. `limitless.jpg`

## Recommended Specs
- Format: JPG or WebP
- Dimensions: 300x450px (2:3 aspect ratio, standard movie poster)
- File size: Keep under 200KB per image for optimal performance
